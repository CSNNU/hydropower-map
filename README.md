# 全国小水电站地图查询 - 总览

本项目包含三个独立版本，分别存放在以下文件夹：

| 文件夹 | 版本 | 登录方式 | 说明 |
|--------|------|----------|------|
| `v1-miniprogram-no-auth/` | 微信小程序 v1.0 | **无**电话权限登录 | 直接打开即可使用 |
| `v2-miniprogram-with-auth/` | 微信小程序 v2.0 | **有**电话权限登录（白名单） | 需输入手机号验证 |
| `html-version/` | HTML网页版 | **无**电话权限登录 | 浏览器直接打开 |

---

## 数据来源与过滤说明

**原始数据：** 2024年全国小水电站基本信息，共 **47,992** 条记录。

**过滤规则：** 以下状态的电站**不显示**在地图上：

| 电站属性 | 数量 | 是否显示 |
|----------|------|----------|
| 已建 | 34,990 | ✅ 显示 |
| 改、扩建 | 424 | ✅ 显示 |
| 在建 | 131 | ✅ 显示 |
| 新建 | 72 | ✅ 显示 |
| 续建 | 11 | ✅ 显示 |
| 拟建 | 5 | ✅ 显示 |
| 核增 | 2 | ✅ 显示 |
| （空属性） | 5,258 | ✅ 显示 |
| **合计显示** | **40,893** | |
| 报废 | 6,921 | ❌ 不显示 |
| 停运 | 2,279 | ❌ 不显示 |
| （其他空） | 1 | ❌ 不显示 |
| **合计不显示** | **9,201** | |

> **最终地图显示：40,893 个水电站**（已排除报废 6,921 个、停运 2,279 个及其他 1 个）

## 各版本说明

### v1-miniprogram-no-auth/（微信小程序 v1.0）
- **无登录限制**，打开即可使用
- 适合公开分享、无需权限控制的场景
- 使用微信开发者工具导入此文件夹即可编译

### v2-miniprogram-with-auth/（微信小程序 v2.0）
- **有手机号白名单验证**，需在 `data/whitelist.js` 中配置允许的手机号
- 适合内部使用、需要权限控制的场景
- 未授权用户无法进入主页面

### html-version/（HTML网页版）
- **无登录限制**，浏览器直接打开 `index.html`
- 基于 Leaflet 地图库，兼容所有现代浏览器
- 数据从 `stations.json` 加载（40,893 个电站）

---

## 技术细节

- 地图坐标：GCJ-02（国测局坐标系），与高德/腾讯地图一致
- 数据压缩：小程序版使用 gzip + 字典编码压缩，减少包体积
- 坐标精度：6位小数（内部存储 ×1000000）
- 装机容量精度：1位小数（内部存储 ×10）
