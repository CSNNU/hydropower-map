# 全国小水电站地图查询

## 数据说明

- **数据来源**：2024年全国小水电站基本信息（共 47,991 条记录）
- **全部包含**：不筛选状态，所有电站均收录
  - 已建：38,106 个
  - 报废：6,921 个
  - 停运：2,279 个
  - 改、扩建：470 个
  - 在建：134 个
  - 新建：64 个
  - 续建：10 个
  - 核增：2 个
  - 拟建：5 个
- **有坐标**：46,389 个（可在地图上显示）
- **无坐标**：1,602 个（仅列表显示）

## 项目结构

```
hydropower-map/
├── v1-miniprogram-no-auth/       # 微信小程序 v1.0（无需登录）
├── v2-miniprogram-with-auth/     # 微信小程序 v2.0（需手机号白名单登录）
└── README.md
```

## 版本说明

| 版本 | 登录方式 | 数据量 | 适用场景 |
|------|---------|--------|---------|
| v1.0 小程序 | 无需登录 | 47,991 个电站 | 快速查询 |
| v2.0 小程序 | 手机号白名单验证 | 47,991 个电站 | 需要用户认证 |

## 技术说明

- 数据使用 gzip 压缩 + base64 编码，主包 < 2MB
- 分块解压加载，避免阻塞 UI
- 坐标系统：GCJ-02（国测局坐标系）
