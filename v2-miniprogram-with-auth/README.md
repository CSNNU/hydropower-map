# 小水电站地图查询 - 微信小程序 v2.0（有电话权限登录）

## 版本信息

- **版本号：** v2.0
- **登录方式：** ✅ 有电话权限登录（手机号白名单验证）
- **适用场景：** 内部使用、需要权限控制的场景

## 权限配置

编辑 `data/whitelist.js` 文件，添加允许的手机号：

```javascript
module.exports = [
  "13800138000",
  "13900139000",
  // 添加更多手机号...
];
```

未在白名单中的手机号将无法进入小程序。

## 使用方法

1. 打开 **微信开发者工具**
2. 导入项目，选择本文件夹作为项目目录
3. AppID 可使用测试号或你自己的小程序 AppID
4. 编译运行，输入白名单手机号登录

## 功能说明

- **手机号白名单登录：** 仅在 `whitelist.js` 中的手机号可使用
- **地图显示：** 基于微信内置地图组件，显示附近水电站标记
- **定位用户：** 自动获取当前位置，支持手动重新定位
- **筛选搜索：**
  - 按名称关键词搜索
  - 按省份/市/县三级联动筛选
  - 自定义搜索半径（1-1000公里）
- **电站详情：** 点击地图标记弹出详情面板
- **联系信息：** 如有联系人和电话，可直接拨打
- **导航前往：** 调用微信内置导航功能

## 数据说明

- **显示电站数：** 40,893 个
- **数据来源：** 2024年全国小水电站基本信息（共 47,992 条）
- **已排除：** 报废电站 6,921 个、停运电站 2,279 个
- **坐标系统：** GCJ-02（国测局坐标系）

## 注意事项

- 数据使用 gzip 压缩 + 字典编码，首次加载需几秒解压
- 需要授权获取地理位置权限
- 地图坐标与高德/腾讯地图一致
- 修改白名单后需重新编译小程序
