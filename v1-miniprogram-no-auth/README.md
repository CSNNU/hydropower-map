# 小水电站地图查询 - 微信小程序 v1.0（无电话权限登录）

## 版本信息

- **版本号：** v1.0
- **登录方式：** ❌ 无电话权限登录（直接打开即可使用）
- **适用场景：** 公开分享、无需权限控制

## 项目结构

```
├── app.js              # 应用入口，加载压缩数据
├── app.json            # 应用配置
├── app.wxss            # 全局样式
├── project.config.json # 微信开发者工具配置
├── data/
│   ├── data.js         # 压缩后的电站数据（Base64 + gzip）
│   ├── loader.js       # 数据加载器
│   └── whitelist.js    # （本版本未使用）
├── utils/
│   └── pako.min.js     # gzip 解压库
├── images/
│   ├── station.png     # 电站标记图标
│   └── user.png        # 用户位置图标
└── pages/
    ├── index/          # 主地图页面
    │   ├── index.js
    │   ├── index.wxml
    │   ├── index.wxss
    │   └── index.json
    └── detail/         # 电站详情页面
        ├── detail.js
        ├── detail.wxml
        ├── detail.wxss
        └── detail.json
```

## 使用方法

1. 打开 **微信开发者工具**
2. 导入项目，选择本文件夹作为项目目录
3. AppID 可使用测试号或你自己的小程序 AppID
4. 编译运行即可

## 功能说明

- **地图显示：** 基于微信内置地图组件，显示附近水电站标记
- **定位用户：** 自动获取当前位置，支持手动重新定位
- **筛选搜索：**
  - 按名称关键词搜索
  - 按省份/市/县三级联动筛选
  - 自定义搜索半径（1-1000公里）
- **电站详情：** 点击地图标记弹出详情面板
- **联系信息：** 如有联系人和电话，可直接拨打
- **导航前往：** 调用微信内置导航功能

## 数据说明

- **显示电站数：** 47,992 个（全部显示，含已建、报废、停运等所有属性）
- **数据来源：** 2024年全国小水电站基本信息
- **坐标系统：** GCJ-02（国测局坐标系）

## 注意事项

- 数据使用 gzip 压缩 + 字典编码，首次加载需几秒解压
- 需要授权获取地理位置权限
- 地图坐标与高德/腾讯地图一致
