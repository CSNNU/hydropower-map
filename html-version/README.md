# 小水电站地图查询 - HTML网页版（无电话权限登录）

## 版本信息

- **版本号：** HTML版
- **登录方式：** ❌ 无电话权限登录（直接打开即可使用）
- **适用场景：** 浏览器访问、无需安装微信

## 使用方法

1. 直接用浏览器打开 `index.html` 即可
2. 推荐使用 Chrome / Edge / Firefox 等现代浏览器
3. 移动端浏览器也可正常使用

## 项目结构

```
├── index.html          # 主页面（单文件，含所有逻辑）
├── stations.json       # 电站数据（40,893个）
├── lib/                # 地图库（本地缓存）
│   ├── leaflet.js
│   ├── leaflet.css
│   ├── MarkerCluster.js
│   ├── MarkerCluster.css
│   └── MarkerCluster.Default.css
└── README.md           # 本文件
```

## 功能说明

- **地图显示：** 基于 Leaflet 开源地图库
- **定位用户：** 点击「获取我的位置」按钮定位
- **搜索半径：** 可自定义 1-1000 公里搜索范围
- **电站详情：** 点击地图标记查看电站信息
- **响应式设计：** 适配桌面和移动端

## 数据说明

- **显示电站数：** 47,992 个（全部显示，含已建、报废、停运等所有属性）
- **数据来源：** 2024年全国小水电站基本信息
- **坐标系统：** GCJ-02（国测局坐标系）

## 注意事项

- 首次加载 `stations.json`（约几MB）需要几秒
- 需要浏览器允许地理位置权限才能定位
- 地图瓦片使用在线服务，需要网络连接
