# 小水电站地图查询 - 微信小程序

## 项目结构

```
miniprogram/
├── app.js              # 应用入口，加载数据
├── app.json            # 应用配置
├── app.wxss            # 全局样式
├── project.config.json # 微信开发者工具配置
├── data/
│   └── stations.json   # 水电站数据 (47991条)
├── images/
│   ├── station.png     # 电站标记图标
│   └── user.png        # 用户位置图标
└── pages/
    ├── index/          # 主地图页面
    │   ├── index.js
    │   ├── index.wxml
    │   ├── index.wxss
    │   └── index.json
    └── detail/         # 电站详情页面
        ├── detail.js
        ├── detail.wxml
        ├── detail.wxss
        └── detail.json
```

## 使用方法

1. 打开 **微信开发者工具**
2. 导入项目，选择 `miniprogram` 文件夹作为项目目录
3. AppID 可使用测试号或你自己的小程序AppID
4. 编译运行即可

## 功能说明

- **地图显示**: 基于微信内置地图组件，显示附近水电站标记
- **定位用户**: 自动获取当前位置，支持手动重新定位
- **筛选搜索**: 
  - 按名称关键词搜索
  - 按省份/市/县三级联动筛选
  - 自定义搜索半径（1-1000公里）
- **电站详情**: 点击地图标记弹出详情面板，显示装机容量、类型、位置等信息
- **联系信息**: 如有联系人和电话，可直接拨打
- **导航前往**: 调用微信内置导航功能

## 注意事项

- stations.json 约11MB，首次加载可能需要几秒
- 需要授权获取地理位置权限
- 地图坐标使用 GCJ-02（国测局坐标系），与高德/腾讯地图一致
